# IT Tech Theme for InstantCMS

A modern, responsive theme designed specifically for IT and technology websites built on InstantCMS.

## Features

- Responsive design that works on all devices
- IT-focused color scheme with blue/tech tones
- Syntax highlighting support for code snippets
- Technology category navigation
- Clean, modern typography
- Optimized for technical content
- Mobile-friendly navigation

## Installation

1. Download the theme package
2. Extract the contents to your InstantCMS themes directory (`/templates/`)
3. The theme folder should be named `it_theme`
4. In your InstantCMS admin panel, go to Settings → Templates
5. Select "IT Tech Theme" as your active template
6. Save the settings

## Customization

### Colors
The theme uses a blue-based color scheme typical for IT websites:
- Primary: #1a2a6c
- Secondary: #2a4d8e
- Accent: #3a6ea5

### Typography
The theme uses a modern font stack:
- Primary: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- Code: 'Courier New', monospace
